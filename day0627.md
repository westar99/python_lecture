- 리스트 컴프리헨션(list comprehension)


```python
"a" in "kiwi"
```




    False




```python
fruits = ['apple','kiwi','mango']
newlists= []
#알파벳 a가 있는 과일만 추출 후 새로운 리스트에 담기
for fruit in fruits:
  #print(fruit)
  if "a" in fruit :
    newlists.append(fruit)
print(newlists)

```

    ['apple', 'mango']
    


```python
#리스트 컴프리헨션으로 표현하면
newlist = [fruit for fruit in fruits if 'a' in fruit]
print(newlist)
```
